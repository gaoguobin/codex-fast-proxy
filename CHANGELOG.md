# Changelog

## 0.2.0

- Rebuilt the Control UI as a lightweight settings-style console with Overview, Providers,
  Requests, Advanced, and Settings pages.
- Added proxy-managed provider add/edit/switch/delete flows with inline checks and masked API-key
  reveal.
- Added Chinese, English, and Japanese UI language support, plus system/light/dark appearance.
- Added update, diagnostics, benchmark, and safe restore/uninstall workflows to the Control UI.
- Hardened update handling for tool-managed install-file deletions from older installs.
- Improved ChatGPT account login compatibility, startup hook summaries, and provider-auth split
  behavior.
- Documented Windows plus macOS/Linux install, update, and uninstall paths.

## 0.1.0

- Initial local proxy and Codex skill workflow for OpenAI-compatible provider traffic.
